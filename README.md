# 🎧 HAP — AirPods Price Prediction

> AI-powered used AirPods price estimator based on real Reddit r/appleswap listings.

---

## 📁 Project Structure

```
hap/
├── main.py                       ← FastAPI server (ML model + prediction API)
├── crawler.py                    ← Reddit r/appleswap scraper
├── reddit_airpods_data_usa.csv   ← Scraped market data (auto-generated by crawler)
├── requirements.txt
└── README.md
```

---

## ⚙️ How It Works

1. **crawler.py** scrapes real listing prices from Reddit r/appleswap
2. **main.py** filters out bundle listings, extracts features from post titles, and trains a Random Forest model on startup
3. The `/predict` endpoint returns an estimated resale price based on generation, condition, and accessories

---

## 🚀 Setup

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. (Optional) Refresh market data
```bash
python crawler.py
```

### 3. Start the server
```bash
uvicorn main:app --port 8000
```

### 4. Test it
Open in browser: **http://localhost:8000/docs**

## 📡 API Reference

### `POST /predict`

**Request:**
```json
{
  "generation": 2,
  "is_new": false,
  "has_applecare": true,
  "is_usbc": true
}
```

| Field | Type | Description |
|---|---|---|
| `generation` | int | 1 = Pro 1st gen, 2 = Pro 2nd gen, 3 = Pro 3rd gen |
| `is_new` | bool | BNIB / Sealed |
| `has_applecare` | bool | AppleCare+ included |
| `is_usbc` | bool | USB-C version |

**Response:**
```json
{
  "predicted_price_usd": 110.0,
  "price_range_low": 101.4,
  "price_range_high": 118.6,
  "currency": "USD",
  "data_source": "Reddit r/appleswap",
  "message": "AppleCare+ adds value. USB-C model priced higher."
}
```

### `GET /model/status`
Returns model stats: data count, MAE, R², feature importance.

### `POST /retrain`
Re-trains the model after updating the CSV.
